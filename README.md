Используйте `docker compose up -d`, чтобы поднять сервер на моём готовом образе, либо используйте его в качестве базового.

Для быстрой загрузке введите IP адрес сервер в `docker-compose.yaml` в переменную IP

После добавления карт и плагинов пишите `docker compose build` для сборки образа.

Для пересборки образа, используйте архив из релиза, там лежат все карты и моды:
https://github.com/evgeniyfimushkin/counter-strike-1.6/releases/download/1.0/counter-strike-1.6.tar.gz
